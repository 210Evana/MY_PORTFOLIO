## 🖼️ How to Add Your Profile Photo

1. Get a photo of yourself (square or portrait works best)
2. Rename it to **`profile.jpg`**
3. Place it in the **same folder as `index.html`**
4. Open `index.html` and find this comment in the HERO section:

```html
<!-- ✅ UNCOMMENT THIS when your photo is ready: -->
<!-- <img src="profile.jpg" alt="Yvonne Nafula Juma" class="hero-photo" ... /> -->
```

5. **Remove the `<!--` and `-->` comment tags** to activate the image
6. Also **delete** (or hide) the placeholder div above it:

```html
<div class="photo-placeholder" id="photoPlaceholder">...</div>
```

That's it — your photo will appear in a styled frame on the right side of the hero section!

> **Tip:** For best results, use a photo that's at least 400×500px, with good lighting and a clear face. JPG, PNG, or WebP all work — just rename to `profile.jpg`.

## 🚀 Deploy to Netlify (Free)

1. Go to [netlify.com](https://netlify.com) → Sign up / Log in
2. Click **"Add new site"** → **"Deploy manually"**
3. Drag & drop the entire project folder onto the page
4. Your site goes live instantly at a `*.netlify.app` URL
5. Optionally set a custom domain in **Site settings → Domain management**

## 🚀 Deploy to Vercel (since you already use it)

1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import the repo → Framework: **Other**
4. Click **Deploy**
